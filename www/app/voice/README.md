# הקלטות "הקול של יפעת"

שימו כאן את קובצי השמע (פורמט `.m4a`):

- `welcome.m4a` — ברכת הפתיחה
- `site1.m4a` … `site11.m4a` — הקלטה לכל אחד מ-11 אתרי המסלול

אם קובץ חסר, האפליקציה מציגה הודעה עדינה ("נקליט ונשבץ בקרוב") — שום דבר לא נשבר.
אחרי הוספת קבצים יש להריץ `npx cap sync` כדי שייכנסו לאפליקציות הנייטיב.
